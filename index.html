what first inversed range will be?

Rules

  • we need time/date limits: starting and end point(e.g. working schedule)
  • start of work time will be +1
  • end of work time will be -1
  • for each initial range we invert markers: -1 for start point, +1 for end point

Let's assume working schedule is from 08:00 till 18:00
